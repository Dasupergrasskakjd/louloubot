# 🤖 Louloubot

## 🚀 Quick Installation

1. **[Download the Extension (ZIP)](https://github.com/bladingpark/louloubot/archive/refs/heads/master.zip)**
2. Unzip the downloaded `.zip` file on your computer.
3. Open your browser (Chrome, Brave, Edge, etc.) and go to `chrome://extensions/`.
4. Enable **Developer mode** (toggle switch in the top right corner).
5. Click on **Load unpacked**.
6. Select the `louloubot-master` folder you just unzipped.
